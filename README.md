# Trainee da iJunior

## Modo de uso

### Para as Semanas 1 e 2, 
digite, no terminal, os comandos:

```bash
npm i

node semana_.js
# _ (underline) deve ser substituído pelo número da semana
```

```bash
npx jest
# Para testar o código
```

### Para a Semana 3, 
digite, no terminal, os comandos:

```bash
npm i

npm install typescript -D

npm run start
# Executa o script "start" em package.json
```
